# Arduino-Control-of-Four-Servo-Motors-

## Overview

This project demonstrates how to control four servo motors simultaneously using an Arduino Uno and the Servo library. The program performs a sweep motion on all four servos for approximately two seconds, then moves them to a 90° position where they remain fixed.

The project was created and tested using Tinkercad Circuits to simulate servo motor control and demonstrate basic Arduino programming concepts.

![Arduino Servo Circuit](circuit .png)

## Components

- Arduino Uno

- 4 × Micro Servo Motors

- Jumper Wires

## Features

- Controls four servo motors at the same time.

- Executes a sweep motion for about two seconds.

- Holds all servo motors at 90° after the sweep.

- Simulated and tested in Tinkercad.
